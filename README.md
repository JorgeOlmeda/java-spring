# java-spring
Projects made in the course Spring Framework 5: Creando web app de cero a experto by instructor Andrés José Guzmán in the platform udemy.

It has been made with Java 8 and Spring Framework 5 and Spring Boot 2.

It has been used Spring MVC, data JPA, Spring Security, Multilanguage i18n, Bootstrap, Thymeleaf and javascript and Mysql 8 .
